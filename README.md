## Kurulum İşlemleri
1- npm install

2- npm install -g nodemon

3- npm start

Sürüm sorunu olursa:

npm uninstall mongodb --save

npm i mongodb@3.6.3

Tarayıcı kısmına localhost:3000 yazarak projeye ulaşabilirsiniz.
